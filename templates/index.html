The bdmerge project

The Biological Data Merge (bdmerge) project seeks to systematically develop and assess methods for integrating data from high-throughput biological experiments across regulatory layers.

Large-scale biology projects generate data using several experimental platforms applied to many cell lines, patient samples, and experimental perturbations such as small-molecules and genetic alterations. However, the reproducibility of discoveries from such data is impeded by differences in experimental and computational procedures. A public repository of paired data sets, along with benchmark methods evaluating cross-type similarity, could assess the consistency of different experimental and computational pipelines. The developed benchmarks operate on two data matrices (samples x features) with either matched samples or matched features and are therefore generally applicable to data in this format.

For a longer descriptions about the project.

The data repository is available through the browse functionality and each entry can be downloaded as data packages or as individual files. Each entry (or data project) is constructed around a pairwise data collection subjected to different computational pipelines. All code for data analysis is available either through analysis script or through the R package {bdmerge}.

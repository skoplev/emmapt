Current challenges

Identifying relationships between cancer proteomic and transcriptomic data

For the upcoming P100 LINCS data release on June 30th 2015, the challenge is to submit rules relating LINCS L1000 measurements to the novel P100 measurements.

Evaluation

Submissions will be scored according to a benchmark metric based on transformed similarities. For more information view technical details. The evaluation data consists of the 6 cancer cell lines: A375, A549, MCF7, NPC, PC3, and TBD subject to 32 chemical perturbations.

Requirements

Submissions must include a data matrix of transformed transcriptomic data with matching experimental conditions. Optionally, transformations of the new proteomic data can be provided as R or Python functions. The submitted function must take as input a proteomic data vector and metadata describing the experimental condition and return a vector representing the transformed observation.

Suggestions

The participants are free to implement the transformations however they prefer using any data and methods. The key problem is how to use available data and prior knowledge, such as interaction databases, to identify new relationships between transcriptomic and proteomic data. The appropriate levels of abstraction for such relations are an open research question and the hope is that challenges such as this can provide important answers to such questions.

Predicting outcomes of proteomic experiments

For the upcoming P100 LINCS data release on June 30th 2015, the challenge is to predict the outcome of those experiments using available data. Such predictions would help to establish data saturation in the LINCS project, that is to estimate to what extent new experiments provide new information about the particular experimental conditions.

Evaluation

Submissions will be evaluated by the mean squared deviation between the submitted predictions and the experimental result. The evaluation is performed at the 3rd data level after appropriate normalizations and aggregate statistics have been calculated. Also, each method will be subjectively evaluated based on scientific relevance. Thus, the used methods must be sufficiently well described and documented so as to enable the evaluation by a scientific expert with limited computational ability.

Requirements

Submissions must include a data matrix containing the predictions. Row and column metadata associated with the submitted matrix must uniquely match to the experimental outcome matrix.

Suggestions

Determining which type of data to use for the predictions is a critical problem. Data with matches to existing P100 experiments are more likely to be useful in this regard.
